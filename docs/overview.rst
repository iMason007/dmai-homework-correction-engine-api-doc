概述
======================================

DMAI作业批改引擎提供教辅标注数据管理(包括新增, 修改, 删除, 查询)和教辅页面批改功能. 需要先导入教辅标注数据, 批改引擎才能更好地批改对应的\
教辅页面. 当前版本只支持数学教辅批改, 题型支持填空题, 选择题, 脱式计算, 解方程, 竖式计算, 表达式计算, 勾圈题, 应用题
